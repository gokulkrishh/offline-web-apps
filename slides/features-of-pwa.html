Features of PWA

  • Responsive: Fits any form factor like mobile, desktop etc,.
  • Works Offline: Using service worker
  • Upto-date: Always up-to date thanks to SW update process
  • Safe (https): Served via TLS
  • Performance: Page loads faster
  • Installable: Add to home screen & no installation is required
  • Background functionality: Like push notifications
  • Native App like: Using application shell architecture (ASA)
